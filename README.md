Exercícios de lógica de programação em C# e fluxogramas para minha faculdade.
São duas listas, cada uma com 10 exercícios, separadas em dois diretórios: [lista01/](./lista01/) e [lista02/](./lista02/).

Rode as listas com `dotnet run`. Cada lista deve ser rodada individualmente.

Os fluxogramas dos exercícios estão nos diretórios [/fluxograma](./lista01/fluxogramas/) de cada lista.

Versão do dotnet: 10.0.104
